# 2D Max Pooling (2x2)

输入 `X` 的形状为 `[N, C, H, W]`，数据类型为 fp32，其中 `H` 和 `W` 均为偶数。

前向使用 `2x2` 窗口、步长 2、无 padding，输出 `Y` 的形状为
`[N, C, H/2, W/2]`：

```text
Y[n,c,i,j] = max(X[n,c,2i,2j], X[n,c,2i,2j+1],
                 X[n,c,2i+1,2j], X[n,c,2i+1,2j+1])
```

反向只对输入 `X` 求梯度。每个窗口的上游梯度只回流到该窗口内最大值所在的一个原始位置，其余三个位置梯度为 0；并列最大值按窗口行主序选择第一个位置。
