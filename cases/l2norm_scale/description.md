# L2 归一化 + 每维度缩放（l2norm_scale）

把一批向量各自 L2 归一化（变成单位长度），再按一个可学习的每维度权重缩放。

## 前向

输入 X[N, D]（N 个向量，每个 D 维，fp32），可学习权重 g[D]（fp32）。

    norm[i]   = sqrt( sum_j X[i,j]^2 + eps )      # 每行 L2 模长（eps 防除零）
    Xhat[i,j] = X[i,j] / norm[i]                  # 归一化到单位长度
    Y[i,j]    = Xhat[i,j] * g[j]                   # 每维度缩放

沿最后一维 D 归一化（每行 [D] 当一个向量）。eps=1e-6。

## 输入

- X: [N, D]，fp32。
- g: [D]，fp32，可学习的每维度权重。

## 梯度

对 X 和 g 求梯度（eps 是标量常数，不可导）。
